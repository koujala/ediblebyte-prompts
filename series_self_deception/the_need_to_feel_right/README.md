# The Need to Feel Right

Prompts from **The Need to Feel Right** in the Self-Deception Series.
Read the full article: [ediblebyte.com/publications/series_self_deception/the_need_to_feel_right](https://www.ediblebyte.com/publications/series_self_deception/the_need_to_feel_right)

The article identifies the most fundamental pattern in the series: the need to be right — not to find what is right, but to be right — operating as the background condition under which the whole process runs. Questions get asked in ways that assume the answer. Challenges get heard as attacks rather than information. This prompt forces the strongest version of the case against you.

## Prompts

- [`steelman_against_yourself.md`](steelman_against_yourself.md) — Ask AI to build the strongest possible case that you are simply wrong — not partially right with caveats, just wrong.
