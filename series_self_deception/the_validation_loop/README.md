# The Validation Loop

Prompts from **The Validation Loop** in the Self-Deception Series.
Read the full article: [ediblebyte.com/publications/series_self_deception/the_validation_loop](https://www.ediblebyte.com/publications/series_self_deception/the_validation_loop)

The article identifies jury shopping: consulting opinions until one arrives that confirms what was already wanted, then declaring the consultation complete. The mind performs this with remarkable efficiency and almost no self-awareness — the intent feels genuinely open, the process feels thorough, the conclusion feels earned. This prompt breaks the loop by forcing the fundamental case against the position.

## Prompts

- [`break_the_validation_loop.md`](break_the_validation_loop.md) — Ask AI for the strongest fundamental case against a position after you have already found agreement.
