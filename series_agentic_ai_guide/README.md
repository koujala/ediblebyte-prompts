# Enterprise Agentic AI: A Practitioner's Guide — Prompts

Companion prompt library for the **Enterprise Agentic AI: A Practitioner's Guide** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_agentic_ai_guide). Each folder maps to one article; each prompt inside is a standalone, reusable template drawn from that article.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into Claude, ChatGPT, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and iterate from there.

For recurring use, paste the prompt into the **Instructions** field of a Claude or ChatGPT *Project* so it becomes a persistent workspace.

## The core philosophy

Every prompt here reflects the central argument of the series: **prove value first, earn complexity later.** Ship a narrow, trustworthy system. Measure it honestly. Use that evidence as the foundation for everything that follows.

The sophistication comes later. The discipline has to come first.
