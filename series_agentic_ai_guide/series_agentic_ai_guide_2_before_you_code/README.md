# Post 2 · Before You Write a Single Line of Code

Prompts from **Article 2** of the Enterprise Agentic AI: A Practitioner's Guide series.
Read the full article: [ediblebyte.com/publications/series_agentic_ai_guide/series_agentic_ai_guide_2_before_you_code](https://www.ediblebyte.com/publications/series_agentic_ai_guide/series_agentic_ai_guide_2_before_you_code)

The decisions made before touching a keyboard determine whether an agent reaches production. This post covers two things that are consistently skipped: defining the agent's job with surgical precision and picking a first use case that gives you a real chance of success.

## Prompts

- [`agent_job_definition.md`](agent_job_definition.md) — Answer three questions in writing before your first sprint. The output becomes your political armor inside the enterprise.
- [`use_case_fitness_check.md`](use_case_fitness_check.md) — Evaluate whether a proposed use case is the right first agent problem, using the five traits and three anti-patterns from the article.
- [`pre_code_checklist.md`](pre_code_checklist.md) — Six-item checklist to run before writing a single line of code. If any item is unchecked, that item is the priority — not the code.
