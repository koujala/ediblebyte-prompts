# Rule-Breaking Scenario

The Dreyfus model says the path from competence to proficiency runs through learning to read situations rather than apply procedures. This prompt generates exactly those situations — cases where a rule-follower gets it wrong and a situation-reader gets it right. Use it to train the judgment that textbooks cannot give you.

```
I know the rules of [SKILL OR DOMAIN — e.g. system design / code review / performance conversations / financial modelling].

Give me a situation where following them exactly would produce the wrong outcome — where I'd need to read the situation rather than apply the procedure.

Make it realistic and specific. After I respond with how I would handle it, tell me what a Stage 4 or Stage 5 practitioner would notice that I might have missed.
```

**How to use the output:** Work through the scenario before reading ahead. Commit to a response, then compare it against what the AI surfaces. The gap between your instinct and the expert framing is exactly the gap you are developing. Repeat with different scenarios in the same domain until the gap closes.
