# Series: Expertise and Mastery — Prompts

Companion prompt library for the **Expertise and Mastery** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_expertise_and_mastery). Each folder maps to one article; each prompt file inside is a standalone, reusable template drawn from that article.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into ChatGPT, Claude, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and iterate until the output fits your situation.

## The central tension

Deep expertise is real — and it carries its own failure modes. These prompts help you catch the traps that expertise builds quietly: the questions that stop, the models that stop updating, the complexity that can't be communicated, the confidence that bleeds into adjacent domains.

AI is used throughout this series as a structured mirror — a way to surface what the expert can no longer easily see about themselves.
