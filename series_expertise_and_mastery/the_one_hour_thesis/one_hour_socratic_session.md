# One-Hour Socratic Session

Use this at the start of a dedicated learning hour. It switches AI from a source of answers into a Socratic teacher — asking questions, testing understanding, correcting mistakes, and adapting to where you are stuck. The same hour spent reading passively will not compound the same way.

```
I want to learn [SKILL — be specific: not "Python" but "writing async functions in Python" / not "leadership" but "giving feedback that actually changes behaviour"].

I have one hour. Challenge me — ask questions, test my understanding, correct me when I'm wrong, adapt when I'm stuck. Don't let me off easy.

Start by asking me what I already think I know about [SKILL], then find the gaps.
```

**How to use the output:** Resist the urge to ask AI to just explain things. When it asks you a question, answer it before reading ahead. When it corrects you, reconstruct the reasoning yourself rather than copying the correction. The discomfort of not knowing is where the learning happens — the prompt is designed to produce that discomfort.
