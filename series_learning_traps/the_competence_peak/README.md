# The Competence Peak

Prompts from the **Competence Peak** article in the Learning Traps series.  
Read the full article: [ediblebyte.com/publications/series_learning_traps/the_competence_peak](https://www.ediblebyte.com/publications/series_learning_traps/the_competence_peak)

The article names the trap hidden in competence itself. The Yerkes-Dodson Law shows that below a challenge threshold, performance stagnates and learning stops. Once you're comfortable, the discomfort that signals falling behind disappears — so accurate self-assessment in a static landscape is not the same as accurate self-assessment in a moving one. Comfort is not the reward for having learned enough. It is the signal that learning has stopped.
