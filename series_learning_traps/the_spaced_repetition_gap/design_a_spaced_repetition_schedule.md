# Design a Spaced Repetition Schedule

Use this when you've learned something you want to retain long-term — not just understand now. The prompt builds a month-long review schedule with retrieval questions harder than recognition: you have to produce the answer, not identify it.

The risk it addresses: the spaced repetition gap. Without reinforcement at increasing intervals, 70% of new material is gone within 24 hours. This prompt implements the science that most educational institutions have ignored for 140 years.

---

## The prompt

```
I want to retain [TOPIC/SKILL] long-term. Here are the key concepts I need to internalise: [LIST THEM — 3-7 specific concepts, not broad categories].

Design a spaced repetition review schedule for the next month — what to review, in what form, at what intervals. For each review point, give me a retrieval question I can use to test myself from memory rather than re-reading the material.

Make the questions harder than recognition: I should have to produce the answer, not identify it.
```

---

**How to use:** Be specific about the concepts — "explain the CAP theorem" rather than "distributed systems." The retrieval questions only work if you attempt them from memory before checking. Set actual calendar reminders for the review intervals; the schedule is worthless without follow-through.

**What this reveals:** Nothing immediately — the value is in running the schedule. By the one-month mark you'll know which concepts are durable and which ones required re-learning at every interval. That pattern tells you where to invest more practice.
