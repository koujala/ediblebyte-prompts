# Evaluate as Day One

Use this when you're continuing a course, career path, project, or commitment that you're not sure is still the right choice — and you suspect the time already invested is influencing the decision to keep going. AI doesn't know what you've already paid. This prompt uses that to your advantage.

The risk it addresses: sunk cost fallacy. The past cost is already gone. This prompt strips it from the evaluation and surfaces what the honest answer is when only the future matters.

---

## The prompt

```
I've been [PURSUING PATH/COURSE/COMMITMENT/STRATEGY] for [LENGTH OF TIME]. Here's what it involves and what I was originally hoping to get from it: [BRIEF DESCRIPTION — what you're doing and what the original goal was].

Evaluate this as if I'm making the decision for the first time today — ignore everything already spent. Tell me directly: is this the right choice now? If it isn't, what specifically should I do instead, and what's the earliest point at which the evidence that this wasn't working was already there?
```

---

**How to use:** Write the description of the path as neutrally as possible — don't frame it as "I've already invested X months, so..." That framing carries the sunk cost into the evaluation. Just describe what it is and what you wanted from it.

**What this reveals:** Whether continuing is a genuine forward-looking decision or a backward-looking one dressed as commitment. The question about the earliest evidence of misfit is particularly useful — if the answer is "three weeks in," the past months have been paid for by the sunk cost, not by future value.
