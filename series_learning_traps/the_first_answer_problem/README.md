# The First Answer Problem

Prompts from the **First Answer Problem** article in the Learning Traps series.  
Read the full article: [ediblebyte.com/publications/series_learning_traps/the_first_answer_problem](https://www.ediblebyte.com/publications/series_learning_traps/the_first_answer_problem)

The article names anchoring — the mind's tendency to grab the first coherent answer to a hard question and begin building around it before the sentence is finished. This affects doctors, judges, investors, engineers, and everyone who believes they're too rigorous to be affected. The cost is usually invisible: not wrong answers, but adequate outcomes where better ones were available.

## Prompts

- [`generate_ten_before_ranking.md`](generate_ten_before_ranking.md) — Break the anchor by generating a full range of genuinely different approaches before evaluating any of them.
