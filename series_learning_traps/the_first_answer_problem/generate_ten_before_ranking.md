# Generate Ten Before Ranking

Use this at the start of a problem — before the first instinct has hardened into a direction. The prompt generates ten genuinely different approaches and withholds ranking until you've read all of them, breaking the anchoring effect of whatever arrived first.

The risk it addresses: anchoring. The first coherent answer to a hard problem lands with disproportionate weight. Every subsequent idea gets unconsciously evaluated against it rather than on its own merits. This prompt delays the anchor from forming until the space has been explored.

---

## The prompt

```
I'm working on [PROBLEM/DECISION/CHALLENGE]. My first instinct is already [WHAT YOU'RE CURRENTLY LEANING TOWARD — be honest].

Give me ten genuinely different approaches — not variations of the same idea. For each, note what type of thinking or discipline it draws from.

Then give me one more: the approach someone would suggest if they rejected the constraints I've taken for granted.

Don't rank them — I want to evaluate each one before any ordering influences me.
```

---

**How to use:** State your first instinct explicitly — this surfaces the anchor so you can watch for it pulling you back. Read all ten before comparing any. The eleventh approach (rejecting your assumed constraints) is often the most useful one; don't skip it.

**What this reveals:** The ceiling set by your first idea, and what's available beyond it. If most of the ten feel like variations of your instinct, prompt again: "Make these more genuinely different — some should come from outside [FIELD/APPROACH]."
