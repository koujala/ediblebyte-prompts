# Learning Traps — Prompt Library

Companion prompt library for the **Learning Traps** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_learning_traps). Each folder maps to one article; each prompt file is a standalone, reusable template drawn directly from that article.

The series names the cognitive blind spots that make people think they're learning when they're not — and shows how to use AI to break out of each trap.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into ChatGPT, Claude, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and run it as written. The prompts are designed to be copy-paste ready; no assembly required.

## The underlying principle

Each prompt in this library is designed to break the passive loop — to turn AI from a source of convenient answers into a source of productive friction. Recognition is not capability. The prompts here are calibrated to test what you actually know, not confirm what you think you know.
