# The Context Collapse

Prompts from the **Context Collapse** article in the Learning Traps series.  
Read the full article: [ediblebyte.com/publications/series_learning_traps/the_context_collapse](https://www.ediblebyte.com/publications/series_learning_traps/the_context_collapse)

The article names the Encoding Specificity Principle: memory is encoded with the context in which it was formed. Retrieval is easier in the same context. A skill learned in a workshop with strangers is encoded with workshop-strangers. Applying it in a real relationship with real stakes requires transfer — a separate capability that requires separate practice. Most training environments don't build transfer. They build performance-in-training, and call it learning.

## Prompts

- [`transfer_to_real_context.md`](transfer_to_real_context.md) — Apply a skill or framework to the specific real situation it needs to function in, including what changes when the conditions aren't controlled.
- [`build_transfer_through_varied_practice.md`](build_transfer_through_varied_practice.md) — Build genuine fluency across different contexts by practising the same skill across varying stakes, relationships, and emotional conditions.
