# Transfer to the Real Context

Use this when you've learned a skill or framework in a controlled setting and now need to apply it in a real situation that looks different. The prompt walks through the specific real context — not the textbook version — and flags where the gap between training and reality will most likely appear.

The risk it addresses: context collapse. Skills encoded in workshops and courses often fail in real situations because retrieval depends on context, and the real context is different in every way that matters: the relationship, the stakes, the emotional charge, the history.

---

## The prompt

```
I learned [SKILL/FRAMEWORK — e.g. "the BATNA negotiation framework", "giving structured feedback using SBI", "the five dysfunctions model"] in a [WORKSHOP/COURSE/CONTROLLED SETTING]. The real situation I need to use it in is: [DESCRIBE THE ACTUAL CONTEXT — the relationship, the power dynamic, the stakes, the history].

Walk me through applying the skill specifically in this context. What changes when the setting is real rather than simulated? Where will I most likely freeze or revert to instinct?
```

---

**How to use:** Be specific about the real context — not "a negotiation" but "a salary conversation with my direct manager after a year where I've been under-levelled and we both know it." The specificity is what produces useful guidance. Generic application of a framework to a real situation is still just theory.

**What this reveals:** The gap between knowing a framework and being able to execute it when the conditions don't cooperate. The question about where you'll freeze is the most important one — knowing your likely reversion point in advance is the difference between catching it and being caught by it.
