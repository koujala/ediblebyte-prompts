# The Tutorial Plateau

Prompts from the **Tutorial Plateau** article in the Learning Traps series.  
Read the full article: [ediblebyte.com/publications/series_learning_traps/the_tutorial_plateau](https://www.ediblebyte.com/publications/series_learning_traps/the_tutorial_plateau)

The article names a specific failure mode in tutorial-based learning: tutorial learning delivers observation and abstraction (stages 2 and 3 of Kolb's cycle) while skipping the stages that build real skill — doing and experimenting. The fluency illusion compounds it: watching an expert solve a problem produces a feeling of understanding that is not the same as being able to solve it yourself. Following someone else's reasoning teaches you to follow. It does not teach you to think.

## Prompts

- [`diagnose_your_plateau.md`](diagnose_your_plateau.md) — Expose the line between real understanding and scaffold dependency with a blank-canvas problem and an honest debrief.
- [`build_without_scaffold.md`](build_without_scaffold.md) — Rebuild a tutorial project from scratch with Socratic hints only, to find out what you actually know vs. what the tutorial was providing.
