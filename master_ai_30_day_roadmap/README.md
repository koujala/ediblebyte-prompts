# Mastering AI — Your 30-Day Roadmap

Day-by-day prompts across three phases: Claude App (learn and produce), Claude CoWork (automate), and Claude Code (build and ship). Every prompt uses `[brackets]` for your specifics.

## Phases

- **Phase 1 — Claude App (Days 1–14):** [`phase1_claude_app.md`](phase1_claude_app.md) — Learning, producing, and capturing knowledge
- **Phase 2 — Claude CoWork (Days 15–21):** [`phase2_cowork.md`](phase2_cowork.md) — File management, web tasks, and weekly automation routines
- **Phase 3 — Claude Code (Days 22–30):** [`phase3_claude_code.md`](phase3_claude_code.md) — Exploring, building, deploying, and reflecting

## Usage

Work through each phase in order. Each prompt is a starting point — iterate beyond the first output. The value compounds when you push past draft one.
