# Phase 2 — Claude CoWork: Automate (Days 15–21)

Prompts for auditing what drains you, managing files, running web research, and building a weekly automation routine.

---

## Day 15 — Audit what drains you

```
Here are 5 tasks I do repeatedly that feel manual and draining: [list them]. For each one, tell me: (1) whether Claude CoWork can handle it, (2) how I'd set it up, (3) how long it would realistically take to configure.
```

---

## Days 16–17 — File and folder management

**Organise a folder:**
```
I have a folder at [path] that's completely disorganised. Look at what's in it and suggest a folder structure that would make sense for this type of content. Then ask me to confirm before you move anything.
```

**Sort files by type:**
```
Go through my [Downloads / Desktop / project folder] and group the files by type — documents, images, spreadsheets, code files, etc. Show me what you'd move where before doing anything.
```

**Clean up duplicates:**
```
Look through [folder path] and identify any files that look like duplicates — same name, similar name with (1) or copy in it. List them and ask me which ones to keep.
```

**Rename files consistently:**
```
The files in [folder] have inconsistent naming — some have dates, some don't, some are all caps. Suggest a consistent naming pattern and show me what the renamed files would look like before you change anything.
```

## Days 18–19 — Web tasks

**Summarise a website:**
```
Go to [URL] and give me a summary of what this site is about, who it's for, and the 5 most important things on the page.
```

**Extract structured data:**
```
Go to [URL — a product page, a directory, a list page] and pull out all the [product names / prices / names / dates] into a clean table.
```

**Research across multiple sources:**
```
I need to understand [topic / company / person / trend]. Search for the most relevant recent information, visit the top 3–4 sources, and give me a structured briefing: key facts, main points of view, anything surprising or contradictory.
```

**Monitor a page for changes:**
```
Go to [URL] and note the current state of [specific section — pricing, job listings, product availability]. I'll ask you to check it again later and tell me what changed.
```

**Compare competitors:**
```
Visit the websites of [Company A], [Company B], and [Company C]. Compare them on: messaging, pricing (if visible), key features they promote, and tone. Give me a table.
```

---

## Days 20–21 — Build a weekly CoWork routine

**Day 20:**
```
Based on the tasks I've given you this week, help me design a weekly CoWork routine. What should I run at the start of each week? What's worth doing daily? What makes sense on demand?
```

**Day 21 — find more opportunities:**
```
I work in [role] and my week typically involves [describe 3–4 regular activities]. What are 3 CoWork tasks I probably haven't thought to try that would save me real time? Walk me through each one.
```
