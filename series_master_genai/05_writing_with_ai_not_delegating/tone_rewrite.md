# Tone Rewrite

Three variations on the same idea: take something you have already written and recalibrate it — for clarity, for a specific audience, or for length. Use whichever fits the moment.

## Variant A — Rewrite for clarity and brevity

```
Rewrite the following for clarity and brevity. Remove anything redundant. Keep the tone professional but human. Maximum [WORD COUNT] words.

---
[PASTE THE TEXT]
---
```

## Variant B — Adjust tone for a specific recipient

```
Rewrite the following for [SPECIFIC RECIPIENT — e.g. "a senior executive who values directness and has limited time"].

Lead with the ask, not the context. Keep it short. Keep it courteous. Do not hedge.

---
[PASTE THE TEXT]
---
```

## Variant C — Match a different register

```
Rewrite the following so it lands with [AUDIENCE — e.g. "a technical peer who will be irritated by corporate language", "a client I have a warm relationship with", "a board that will not have read the pre-read"].

Preserve the substance. Change only the register: vocabulary, sentence length, and level of formality. Flag anywhere the underlying argument itself is weak and would need more than a tone change to work.

---
[PASTE THE TEXT]
---
```

**Tip:** Always read the rewrite aloud before you send it. If it sounds wrong when spoken, it reads wrong when received.
