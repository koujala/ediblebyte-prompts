# Adjustable Tutor: The Complexity Dial

AI is a tutor with an adjustable complexity dial. "Explain this like I'm 10" and "explain this like I'm a graduate student in the field" produce genuinely different responses to the same question. Use that dial deliberately.

## Prompt — First pass (from zero)

```
Explain [CONCEPT] as if I have never encountered it before.

Start with a simple analogy from everyday life. Then build from the analogy to the actual idea.

Avoid jargon. Where a technical term is unavoidable, define it the first time you use it.

End with one sentence that a newcomer should be able to repeat back in their own words after reading this.
```

## Prompt — One level deeper

Once the basics have landed, use:

```
Good. Now explain [CONCEPT] again, assuming I now understand [LIST WHAT YOU NOW UNDERSTAND].

Go one level deeper:
- introduce the nuance I should know at this stage
- flag the common misconception someone at my current level is most likely to hold
- give me one example where the intuition I just built would actually mislead me
```

## Prompt — Expert level

When you are ready:

```
Now explain [CONCEPT] at the level you would use with someone working in this field professionally.

Include:
- the subtleties or edge cases a beginner wouldn't know to ask about
- where this idea sits within the broader landscape of related concepts
- one open or contested question in this area — something experts still disagree on
```

**Tip:** Don't skip the steps. Jumping straight to "expert level" on a topic you don't actually have the foundation for produces explanations that sound impressive and teach you nothing. The dial is most useful turned one notch at a time.
