# Series Master: GenAI Prompts

Companion prompt library for the **Gen AI Mastery** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_master_genai). Each folder maps to one article; each prompt inside is a standalone, reusable template drawn from that article.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into ChatGPT, Claude, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and iterate until the output matches what you actually need.

For recurring use, consider pasting the prompt into the **Instructions** field of a ChatGPT or Claude *Project* so it becomes a persistent workspace (see [`../adaptive_tutor/README.md`](../adaptive_tutor/README.md) for Project setup steps).

## The 80 / 20 discipline

Every prompt here reflects the same underlying principle from the series: **let AI handle the 80%** — drafting, synthesis, structuring, routine processing — so that your attention, judgment, and accountability can go to the **20% that only you can bring**.

AI proposes. You decide.
