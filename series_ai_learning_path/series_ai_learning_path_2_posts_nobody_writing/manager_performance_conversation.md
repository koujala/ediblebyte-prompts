# Manager — Preparing for a Hard Conversation

From Post 2 of the Your AI Path series: the example prompt showing what Phase 1 looks like for a manager.

The problem this solves: managers regularly have to deliver feedback, navigate conflict, and handle underperformance. These conversations are consequential. Most managers go in underprepared because preparation itself feels difficult. This prompt provides a framework and key phrases before you walk into the room — the judgment stays human, the blank page is gone.

## The prompt

```
I need to have a performance conversation with a team member who is consistently missing deadlines.

They're technically strong but seem disengaged. I want to be direct but fair, focus on the impact not the intention, and leave the conversation with a clear improvement plan.

Help me structure this conversation and give me the key things to say.
```

## How to use

Replace the specific situation with your own — describe what's happening, how long it's been going on, and what outcome you're aiming for. Mention the team member's likely reaction if you can anticipate it — this lets the model help you prepare for different scenarios (defensive, shut down, emotional).

Use the output as a briefing before the conversation, not a script during it. The specific words will need to be yours; the structure and key messages are where the model adds value.
