# Scientist — Literature Summary from Abstracts

From Post 2 of the Your AI Path series: the example prompt showing what Phase 1 looks like for a researcher.

The problem this solves: PhD students and researchers can spend 40–60% of their time on literature review before the real work begins. A summary that would take a full day takes 15 minutes with this prompt — the researcher still validates and applies their expertise, but they start from a structured draft instead of a blank page.

## The prompt

```
Here are 8 abstracts from papers on CRISPR-Cas9 efficiency in plant cells.

[paste abstracts]

Summarise the key findings, note where results conflict, and identify what questions remain unanswered based on these papers.
```

## How to use

Paste the actual abstracts into the prompt — don't summarise them yourself first. Number them clearly (Paper 1, Paper 2, etc.) so the model can reference them when noting conflicts. Specify your field and the purpose of the review (literature review section, grant proposal, seminar prep) so the tone and depth are appropriate.

Always validate the output against the source papers. The model can misread nuance in scientific language or overstate consensus where the papers actually disagree.
