# Marketer — Ad Copy Variants

From Post 2 of the Your AI Path series: the example prompt showing what Phase 1 looks like for a marketer.

The problem this solves: marketers spend more time generating content than thinking about strategy. The goal here is not to automate more content generation — it's to get to strategy faster by having options to react to rather than a blank brief.

## The prompt

```
I'm launching a B2B SaaS product for logistics companies. The core value proposition is reducing manual data entry by 70%.

Give me 10 ad copy variants for LinkedIn — some leading with the pain, some with the outcome, some with a specific metric. Keep each under 150 characters.
```

## How to use

Replace the product description and value proposition with your own. Specify the channel (LinkedIn, Meta, Google, email) since copy length and tone vary significantly. Ask for variants that lead with different angles — pain, outcome, social proof, a specific number — so you have a range to react to and select from.

The marketer's job is to react, select, and refine. Ten options in 45 seconds means more time thinking about which approach is right, not whether to write anything at all.
