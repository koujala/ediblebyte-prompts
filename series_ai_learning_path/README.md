# Series: Your AI Learning Path — Prompt Library

Companion prompt library for the **Your AI Path** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_ai_learning_path).

The series traces the full AI adoption journey across three phases — Basics, Integration, Automation — for 13 different professional roles. Only the posts below contain concrete, copy-paste-ready prompts.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into ChatGPT, Claude, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, then iterate. The first output is a draft, not a final result.

For recurring use, paste the prompt into the **Instructions** field of a ChatGPT or Claude *Project* so the context persists across sessions.
