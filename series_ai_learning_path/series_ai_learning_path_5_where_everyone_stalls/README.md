# Post 5 · Where Everyone Stalls — Fix Prompts

Prompts from **Post 5** of the Your AI Path series.
Read the full article: [ediblebyte.com/publications/series_ai_learning_path/series_ai_learning_path_5_where_everyone_stalls](https://www.ediblebyte.com/publications/series_ai_learning_path/series_ai_learning_path_5_where_everyone_stalls)

Most people get through the first week of the Basics Phase fine. Then they hit the same five walls — in the same order. This post names each wall and gives the corrective move. The prompts here are the "fix" examples from the article — the before/after that shows what the corrected version looks like.

These prompts apply across all 13 roles. The walls are not role-specific.
