# Wall 3 Fix — Stay in the Basics Phase: Name Your Three Tasks

From Post 5 of the Your AI Path series: the corrective move for jumping to integration before building fluency.

**The stall:** You've used the Basics Phase for about a week. You start exploring integrations — connecting AI to Notion, automating email, building custom GPTs. Three days later you've spent six hours on setup and haven't used the core tool for anything that actually matters.

**The problem:** The Basics Phase builds fluency. Fluency means knowing — from experience — what the model is good at, what it's bad at, and which of your friction points it genuinely solves. Without that foundation, Integration Phase complexity just creates confusion.

## The fix prompt

Use this to reset and get specific:

```
What are the three things in my week that I consistently procrastinate on, rush, or wish I could skip?
```

Answer that question yourself — don't ask the model. Then use the Basics Phase only on those three tasks for the next three weeks.

**Example answers by role:**

- Project manager: status reports, meeting notes, stakeholder emails
- Teacher: lesson planning, assessment creation, parent communications
- Sales professional: call prep, follow-up emails, CRM updates

## The rule

Stay in the Basics Phase until it genuinely plateaus. That takes weeks, not days.

The signal that you're ready for integration: you're hitting a recurring task where you keep having to rephrase the same prompt because the context resets each time. That's when integration — saving the context permanently — becomes the right move. Not before.
