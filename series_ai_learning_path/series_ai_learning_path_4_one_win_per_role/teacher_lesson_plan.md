# Teacher — Differentiated Lesson Plan

The highest-impact Basics Phase move for a teacher: a fully differentiated lesson plan in minutes instead of hours.

**What changes:** 3 hours of prep becomes 20 minutes of reviewing and adjusting.

## The prompt

```
I'm teaching a [subject] class to [age/year group] students. Topic: [topic name].
I have mixed ability levels — some advanced, some on grade level, some struggling readers.

Create a 45-minute lesson plan that includes:
- A warm-up question to activate prior knowledge (5 min)
- Three differentiated activity versions for the same concept (low/mid/high)
- A closing check-for-understanding question
- One homework task

Tone should be practical, not academic. Format it so I can hand it to a substitute teacher if needed.
```

## How to use

Fill in `[subject]`, `[age/year group]`, and `[topic name]` with your specifics. If your class has specific constraints — particular curriculum standards, limited equipment, unusually large or small group — add those before the activity list.

Review the output for anything that doesn't match your students' context. The model won't know your specific students — adjust the differentiation levels based on what you know about the actual gap in your room.

Iterate: if the warm-up question is too easy or the high-ability activity doesn't push far enough, reply with one specific instruction and run it again in the same chat.
