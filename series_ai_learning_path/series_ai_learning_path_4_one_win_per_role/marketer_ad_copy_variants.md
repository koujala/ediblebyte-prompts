# Marketer — Ad Copy Variants

The highest-impact Basics Phase move for a marketer: 10 ad copy variants in under 60 seconds — different angles, tones, and hooks to react to.

**What changes:** Strategy time replaces generation time. React to options instead of staring at a blank brief.

## The prompt

```
I'm running a [channel: LinkedIn/Meta/Google] campaign for [product/service].
Target audience: [one-sentence description].
Core value proposition: [what it does and why it matters].
Goal of the ad: [awareness/click/sign-up/download].

Give me 10 ad copy variants. Vary the approach:
- Some leading with the pain
- Some leading with the outcome
- Some using a question hook
- Some with a specific number or stat
- Some with social proof framing

Keep each headline under 60 characters. Each body under 150 characters.
```

## How to use

Fill in `[channel]`, `[product/service]`, `[target audience]`, and `[goal]` with your specifics. The channel matters — LinkedIn copy reads differently from Meta or Google.

The value here is volume for selection, not volume for volume's sake. Review the 10 variants and identify which angle you want to test first. Discard the rest. If none feel right, reply with "the pain angle isn't landing — try three more versions that lead with the outcome only."

Use this at the start of a campaign brief, before you've committed to an approach. The time saving is in getting to a testing hypothesis faster.
