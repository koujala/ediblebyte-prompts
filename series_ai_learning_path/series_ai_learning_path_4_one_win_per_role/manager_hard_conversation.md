# Manager — Preparing for a Hard Conversation

The highest-impact Basics Phase move for a manager: a framework for the hard conversation — before you walk in the room.

**What changes:** You stop going into hard conversations underprepared. The judgment stays yours — the blank page is gone.

## The prompt

```
I need to have a [performance / behaviour / role change / redundancy] conversation with a team member.

Context:
- The situation: [describe specifically what's happening and how long it's been going on]
- What I want to achieve: [the outcome you're aiming for]
- What I want to avoid: [the failure modes — what could go wrong]
- Their likely reaction: [how you think they'll respond]

Give me:
1. How to open the conversation (first 2-3 sentences)
2. The core message I need to land — in plain language
3. How to respond if they get defensive
4. How to respond if they shut down
5. How to close with clarity on next steps

I want to be direct and fair. The conversation should be about the impact, not the intent.
```

## How to use

Fill in `[performance / behaviour / role change / redundancy]` with the type of conversation. The more specific you are in the Context section, the more useful the output — describe what's actually happening, not a sanitised version.

The "likely reaction" field is particularly valuable: it lets the model help you prepare for the scenario that worries you most, not just the average case. If you're genuinely unsure how they'll react, say so and ask the model to give you frameworks for two or three different scenarios.

Use the output as a briefing before the conversation, not a script during it. The opening sentences are a starting point — your actual words need to sound like you.
