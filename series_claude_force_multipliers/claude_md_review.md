# CLAUDE.md Review

Audit your CLAUDE.md against your actual codebase — surface what's missing, stale, or at the wrong level (managed / user / project / local).

```
Review my CLAUDE.md: [paste content]. Given this codebase: [describe architecture and stack]. Tell me what's missing, what's stale, and what's at the wrong level. What would you still need to ask me about in a fresh session?
```

**When to use:** When starting a new project, after significant architectural changes, or whenever Claude keeps asking questions that should already be answered by context.
