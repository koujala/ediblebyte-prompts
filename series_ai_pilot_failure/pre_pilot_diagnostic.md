# Pre-Pilot Diagnostic

Run the 8-question assessment before writing a line of code. Covers: business outcome, ownership, data readiness, path to production, economics, adoption, and strategic alignment.

Score 8: ship it. Score 6–7: fix the two gaps. Below 4: stop and fix the organisation first.

```
Run the pre-pilot diagnostic for this AI initiative: [describe use case, current team structure, data situation, and planned approach]. Score each of the eight questions and give me specific, actionable fixes for every question scoring 0 — including what needs to be true before we can score 1.
```

**When to use:** Before any AI pilot starts. If you score below 6, you're not ready to pilot — you're ready to prepare.
