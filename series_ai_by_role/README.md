# AI By Role — Prompt Library

Companion prompt library for the **AI By Role** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_ai_by_role). Each post covers one professional role; each prompt is a standalone, reusable template drawn from that article.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into ChatGPT, Claude, or any LLM of your choice — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and iterate until the output matches what you actually need.

For recurring use, consider saving the prompt into the **Instructions** field of a ChatGPT or Claude *Project* for a persistent workspace.

## The underlying pattern

Every post in this series follows the same structure: **Basics → Integration → Automation**. The prompts here cover the Basics level — the highest-ROI moves you can make this week with one frontier model and no workflow changes.

AI handles the overhead. The judgment stays yours.
