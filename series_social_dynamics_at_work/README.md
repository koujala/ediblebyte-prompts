# Social Dynamics at Work — Prompt Library

Companion prompt library for the **Social Dynamics at Work** series on [ediblebyte.com](https://www.ediblebyte.com/publications/series_social_dynamics_at_work). Each folder maps to one article; each prompt file inside is a standalone, reusable template drawn from that article.

## How to use these prompts

Every prompt file is self-contained. Copy its contents into Claude, ChatGPT, or any LLM — fill in the bracketed `[PLACEHOLDERS]` with your specifics, and iterate until the output matches what you actually need.
